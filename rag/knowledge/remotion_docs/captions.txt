Remotion text captions using AbsoluteFill for positioning.

Example:
import { AbsoluteFill } from 'remotion';

<AbsoluteFill style={{ justifyContent: 'flex-end', alignItems: 'center', paddingBottom: 60 }}>
  <div style={{ color: 'white', fontSize: 48, fontFamily: 'sans-serif', textShadow: '0 2px 8px rgba(0,0,0,0.6)' }}>
    {captionText}
  </div>
</AbsoluteFill>

AbsoluteFill stretches to fill the full frame, letting you position captions
using flexbox instead of manual pixel coordinates.
